# 🧠 AI Medical Image Diagnosis System

## 🚀 Overview
This project is an AI-powered medical image analysis system that detects Pneumonia from chest X-ray images using Deep Learning.

It simulates how AI can assist doctors in faster and more accurate diagnosis through automated image analysis.

---

## 🎯 Features
- 📤 Upload chest X-ray images
- 🧠 AI-based disease detection
- 📊 Confidence score & probability analysis
- 🎯 Severity indicator (Low / Medium / High)
- 🧾 Medical insights explanation
- 📥 Downloadable AI report
- 💎 Premium interactive UI (Streamlit dashboard)

---

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit
- Matplotlib / Pandas

---

## 📂 Dataset
- Chest X-ray Pneumonia Dataset (Public dataset from Kaggle)

---

## 🧠 Model
- Transfer Learning using **MobileNetV2**
- Input size: 224x224
- Output: Binary classification (Normal / Pneumonia)
